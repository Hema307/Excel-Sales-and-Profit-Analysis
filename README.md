# Sales & Profit Analysis Dashboard (Excel)

## Project Overview
This project is an interactive Excel dashboard designed to analyze sales and profit performance across different years, months, categories, and states.

## Dataset Details
The dataset contains:
- Order Date
- Customer Name
- State
- Category & Sub-Category
- Sales, Quantity, Profit
- Month & Year

## Tools Used
- Microsoft Excel
- Pivot Tables
- Formulas Used
- Pivot Charts
- Slicers
- Dashboard Design

## Key Analysis
- Total Sales and Total Profit
- Profit trend by Year
- Sales by Category and Sub-Category
- Top 5 States by Sales
- Top 5 Customers by Profit
- Monthly Sales Trend
- Customer Count by Year

## Dashboard Features
- Interactive slicers (Year, Month, Category)
- KPI cards for quick insights
- Dynamic charts connected to pivot tables

## Conclusion
This dashboard helps business users quickly understand sales performance, profitability trends, and customer behavior for better decision-making.
